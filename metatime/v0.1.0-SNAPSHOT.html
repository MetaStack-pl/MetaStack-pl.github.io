Abstract: Time formatting library for Scala and Scala.js

Introduction

MetaTime is a time formatting library for Scala and Scala.js.

Installation

Add the following dependencies to your build configuration:

libraryDependencies += "pl.metastack" %%  "metatime" % "0.1.0-SNAPSHOT"  // Scala
libraryDependencies += "pl.metastack" %%% "metatime" % "0.1.0-SNAPSHOT"  // Scala.js

Manual

Run the following command to generate the manual:

$ sbt manual/runMain pl.metastack.metatime.manual.Manual

Deploy it with:

$ sbt manual/runMain pl.metastack.metatime.manual.Deploy
