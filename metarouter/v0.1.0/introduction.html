Introduction

MetaRouter is a routing library for Scala and Scala.js.

Installation

Add the following dependencies to your build configuration:

libraryDependencies += "pl.metastack" %%  "metarouter" % "0.1.0"  // Scala
libraryDependencies += "pl.metastack" %%% "metarouter" % "0.1.0"  // Scala.js
